Как использовать скрипт MAX* Killer в вашем браузере (работает почти для всех браузеров)
Приветсвтую, рассказываю как использовать скрипт в вашем браузере. Для примера использую браузер Chrome

Устанавливаем Tampermonkey** - расширение для создания пользовательских скриптов
ССЫЛКА

2. После скачивания нажимаем правой кнопкой мыши по расширению -> Управление расширениями -> Разрешить пользовательские скрипты (разрешаем)

3. Заходим на мой гитхаб и открываем текстовый документ max.killer.via.tampermonkey. Копируем код (ПОЛНОСТЬЮ)

4. Жмем ЛКМ по tampermonkey и создаем новый скрипт. вместо шаблонного кода вставляем скопированный ранее код и сохраняем скрипт.

5. Открываем ссылку на плохиш мессенджер и смотрим результат

ПРОБЛЕМЫ И РЕШЕНИЯ:
1. Плохиш мессенджер открывается после установки скрипта (частая проблема)
Решение: 
 1. Ctrl + F5 на странице Плохиш мессенджера.
 2. Если 1 не помогает, то отключаем безопасный DNS сервер в браузере (через настройки вашего браузера соответсвенно)
 3. Если 2 не помогает, то сбрасываем DNS или временные файлы через командную строку (cmd). Открываем cmd (Win+R -> cmd). В командной строке пишем ipconfig/flushdns.
 4. Если первые 3 совета не помогают - свяжитесь со мной через эл.почту (zeroxone.st@gmail.com)

* - Плохиш мессенджер
** - Обязательно к установке
